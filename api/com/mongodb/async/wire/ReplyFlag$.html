    Set when the server supports the AwaitData Query Option.

    Set when the server supports the AwaitData Query Option. If it doesn't, a client should sleep a little between getMore's on a Tailable cursor. MongoDB 1.6+ support AwaitData and thus always sets AwaitCapable
